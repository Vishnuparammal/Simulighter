# SIMULIGHTER

* Abstract:
	The project aims at creating a software to simulate the behaviour of a ray of light in an optical system.

* Features:
	* laser source
	* curved mirror
	* plain mirror
	* convex lens
	* grid
	* undo
	* redo

* Language:
	* HTML
	* CSS
	* Javascript

* Libraries:
	* p5.js

* References:
	* https://ricktu288.github.io/ray-optics/
	* https://p5js.org/reference/
